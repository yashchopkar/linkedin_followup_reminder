# LinkedIn Follow-up Reminder

A Chrome Extension that highlights LinkedIn chats requiring follow-up after selected number of days.

## Features

- Red highlight for overdue conversations
- Badge counter showing number of follow-ups
- Custom threshold (2, 3, or 5 days)
- Searchable follow-up list
- Open chat directly from popup

## Installation (Manual)

1. Download this repository as ZIP  c
2. Extract the folder
3. Open Chrome and go to `chrome://extensions`
4. Enable **Developer Mode**
5. Click **Load unpacked**
6. Select the extracted folder

## How It Works

The extension scans LinkedIn messaging page and highlights chats where the last message is older than selected days.

All data is stored locally in your browser.

## Privacy

This extension does not collect or send any data.
All processing happens locally.
